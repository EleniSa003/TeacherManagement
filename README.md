Teacher Management System

Java desktop application developed using Java Swing, MySQL and JDBC.

Description

The application allows the management of teachers and courses through a graphical user interface. Data are stored in a MySQL database and accessed using JDBC.

Features

- Search teachers by surname
- Add new teachers
- Update teacher information
- Delete teachers
- View assigned courses
- Assign courses to teachers
- MySQL database integration using JDBC

Technologies

- Java
- Java Swing
- MySQL
- JDBC

## Project Structure

- StartFrame.java – Main application window
- ResultsFrame.java – Teacher search results
- InsertFrame.java – Insert new teacher
- CoursesFrame.java – Course management
- DatabaseConnection.java – Database connection
- Program.java – Application entry point
